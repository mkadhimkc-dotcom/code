# 🎀 Girly Shape & Core Definition

A simple, mobile-friendly workout tracking web app designed for glutes, back, core, and cardio training.

## Features
- Body-part based training (Glutes, Back, Core, Cardio, Wrist Care)
- Set tracking with persistence
- 4-week program schedule
- Rest timer
- GIF sliders
- iPhone home screen support

## Run
Open index.html in browser.
