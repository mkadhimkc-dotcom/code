# 🎀 Girly Shape & Core Definition

A simple, mobile-friendly workout tracking web app designed for glutes, back, core, cardio, and wrist-care training.

## Features

- Body-part based training sections
- Data-driven workout cards from `data/workouts.json`
- Set tracking with localStorage and optional Supabase persistence
- 4-week program schedule
- Rest timer
- GIF sliders
- iPhone home screen friendly layout

## Run Locally

The app fetches workout data from `data/workouts.json`, so run it from a local static server:

```sh
node scripts/dev-server.js
```

Then open `http://localhost:8000`.
